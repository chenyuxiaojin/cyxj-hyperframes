# claude-edit-intro (23s · 1920x1080 · 60fps)

XCYJ 分享会的第 2 条视频 —— Claude Edit 教程片头。23 秒短片头，6 个 sub-composition。

## 做了什么

- 总时长 23.17 秒，6 个 sub-composition
- 60fps 渲染
- 纯演示无录屏

## 用了什么技术

- HyperFrames CLI（60fps）

## 怎么复用

适合做"工具功能介绍片头" 的 20-30 秒短视频。技术要点：
- 6 个 beat × ~4 秒，节奏快
- 60fps 适合需要丝滑动效的短片头
- 整片端到端做完只需要几小时
