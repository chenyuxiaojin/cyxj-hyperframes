# Official Template Review

## Audience Judgment

The first Codex cut works as a proof of pipeline, but the cards feel too much like generic glass UI. For this topic, the viewer needs a stronger title-card language and clearer educational structure.

## Templates Checked

- `warm-grain`: warm and editorial, but too lifestyle for a dark talking-head tech tutorial.
- `play-mode`: energetic and elastic, but too playful for Claude Code production teaching.
- `swiss-grid`: strong fit. Structured, precise, technical, readable.
- `kinetic-type`: strong fit for the first 8 seconds. It creates immediate title-card authority.
- `decision-tree`: strong fit for explaining natural language -> Claude Code -> HyperFrames.
- `product-promo`: useful only as a multi-scene pacing reference; not the visual language.
- `nyt-graph`: too data-story oriented for this video.
- `vignelli`: strong but portrait-first and too poster-like for this horizontal talking-head footage.

## Chosen Direction

Use `kinetic-type` for the hook, `swiss-grid` for cards, and `decision-tree` for the process diagram. Remove oversized glass panels that cover the face. Keep cards in the left/lower safe zones and let the speaker remain the main subject.
