# Assets Index

素材真实目录：

```text
../../assets/
```

当前内容在 `../../assets/`。使用素材时只在工程内复制或引用需要的文件，不要把整个素材目录盲目塞进新项目。

## 使用原则

1. 做视频前先查 `../../assets/` 是否已有 logo、纹理、音效、人物素材、截图素材。
2. 新工程需要长期保留的素材，复制到该工程自己的 `assets/` 中，避免渲染时依赖会移动的外部路径。
3. 临时调试图放工程 `debug-shots/` 或 `/tmp/`，不要放顶层。
4. 大视频、大音频、渲染产物不要进 git；按子工程规则处理。

## 跨端一致性

Codex 和 Claude Code 都只需要看到相同的相对路径：

```text
../../assets/<file>
```

不要写 Claude 专属环境变量路径。
