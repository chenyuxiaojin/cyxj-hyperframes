# SCRIPT — N1-1 陈与小金 Website Teaser

**总时长**: 15 秒
**配音**: ⚠️ **无 voiceover**（明天分享会现场演讲者口播）。本 SCRIPT 是 **visual narrative 剧本**，描述每 beat 的屏幕文字 + 视觉动作，供 STORYBOARD 引用。
**调性**: 黑色都市 + 烟雾 + 霓虹点缀（沿用站点 DNA），慢呼吸节奏，沉默有故事感
**结构**: Hook → Identity → Platforms → Outro（4 beat）

---

## Beat 1 — Hook (0–3s)

**画面**:
全黑底 → 白色烟雾 tile 从两侧 fade in 涌入 → 中央"陈与小金"四个大字 Noto Serif SC 900 米白色 (#E8E8E6) fade in，scale 1.05 → 1.0 ease-out。

**屏幕文字**:
- `陈与小金` （Noto Serif SC 900, 200px, 米白居中）

**意图**: 3 秒内观众必须知道"这是个有调性的 IP 主页"——靠中文衬线大字 + 烟雾建立气质。

---

## Beat 2 — Identity (3–7s)

**画面**:
"陈与小金" 大字保持 → 下方副字"非程序员 · Claude Code Practitioner" Inter 500 浅灰 (#C7C7C7) 从下方滑入（y: +20 → 0, opacity 0 → 1）→ 同时左侧紫 `cube.webp`、上方蓝 `moon.webp`、右侧紫 `smile.webp` 三个 3D 霓虹小物按 0.2s stagger 依次浮现，每个伴随轻微 scale bounce（0.6 → 1.0, ease back.out）。

**屏幕文字**:
- `陈与小金` （继续，从 200px 收缩到 140px 上移让位）
- `非程序员 · Claude Code Practitioner` （Inter 500, 36px, 浅灰，居中下方）

**意图**: 用 3 个霓虹 3D 小物建立"非程序员/独立创作者/小工具"的视觉语义，米白大字交代身份。

---

## Beat 3 — Platforms (7–12s)

**画面**:
中央大字组合（陈与小金 + 副字 + 3 物）一起向上滑出（y: -800, ease-in） → 同步从右侧滑入 3 张平台主页截图（`douyin.webp` / `bilibili.webp` / `wechat.webp`），按 0.15s stagger 横向排列，每张 480×640，圆角 24px，#18181B 背景框 → 每张下方浮出 caption（Inter 600, 24px, 米白）。烟雾 tile 继续在底层飘。

**屏幕文字**:
- `抖音 · 408 粉` （第一张下方）
- `B 站 · Lv4 · 21 条` （第二张下方）
- `视频号 · 主阵地` （第三张下方）

**意图**: 用真实平台主页截图直接证明"陈与小金"在三个平台都活跃 = 非程序员能用 Claude Code 真的产出内容并发布。

---

## Beat 4 — Outro (12–15s)

**画面**:
3 张平台卡 fade out（opacity 1 → 0, scale 1 → 0.92, ease-in）→ 中央霓虹粉 (#FF6BB3) 大字 `demo.xiaochens.com` Inter 900 fade in + 0.4s shimmer 扫光（CSS gradient mask sweep） → 全程 grain-overlay 加深 opacity 到 0.18。

**屏幕文字**:
- `demo.xiaochens.com` （Inter 900, 96px, 霓虹粉, shimmer 扫光）

**意图**: 站名 outro，让观众记住 URL 可访问。霓虹粉是站内唯一 CTA 色，与"联系我"按钮呼应。

---

## 节奏摘要

| Beat | 起止 | 时长 | 主元素 | 上屏字数 |
|---|---|---|---|---|
| 1 Hook | 0-3s | 3s | "陈与小金" 大字 + 烟雾 | 4 字 |
| 2 Identity | 3-7s | 4s | 副字 + 3 霓虹 3D | 4 + 16 字 |
| 3 Platforms | 7-12s | 5s | 3 平台卡 + caption | 3 行 caption |
| 4 Outro | 12-15s | 3s | URL + shimmer | 1 URL |

总: 15 秒，4 beat，**密度高但每 beat 都有 1 个清晰 visual focal point**——避免一帧塞太多。
